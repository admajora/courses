### 2.1 Basic Selectors, Relational Selectors

- ul li
  - descendant selector
  - matches nested li's
- ol > li
  - child selector
  - matches li's in ol but not nested ul
- li.myClass + li
  - adjacent sibling
- li.myClass ~ li
  - general sibling selector
  - matches later siblings, but not nested
